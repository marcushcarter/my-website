Setting Up Visual Studio Code With GCC and MINGW

In this lesson, you will learn what Visual Studio Code does, what a code editor is, how to setup Visual Studio Code and how to run your first lines of code.

Downloading Visual Studio Code

The first step is to download the Visual Studio Code installer for the operating system you are working with. I will be using Windows 11, so I will download the Windows version, but this procedure works for all operating systems. Once the installer is finished downloading, save and open it.

When the installer opens, accept the licensing agreement and continue. On the next page it will give you a list of options to select. All are recommended, but you can pick and choose which ones you would like to select except for Add to PATH which is REQUIRED. You will then click next, install and wait for VS Code to finish installing. When it is done installing, launch VS Code.

We now have VS Code Installed but we are going to install some extensions to help you write your code. On the left hand side of the window, find the extensions tab and look up C/C++. Here we are looking for these four extensions all by Microsoft: C/C++, C/C++ Extension Pack, C/C++ Themes and Makefile Tools. Click on each option and when the page that pops up, click 'install'.

We are done installing VS Code, But in the current state it is just a code editor and can't actually run code. To run code with VS Code, we need to set up GCC and MinGW.

Using GCC with MinGW

The first step to setting up gcc and MinGW is to install the MinGW-w64 compiler toolchain that compiles C/C++ code for us. We will do this by MSYS2 which is a software on Windows that will update MinGW whenever it is available. You can download the newest version of the MSYS2 installer or you can use this direct link to the installer. Once the installer launches, click continue until MSYS2 beigns downloading and once it finishes, launch the MSYS2 UCRT64 Terminal (there should be an option to open it in the installer).

Now before we continue, we must check what type of system our device is using because if you install the wrong type of MinGW, your compiler will not work. To do this, open settings and go to System > About > Device Specifications > System Type and check yours.

Depending on what system type you have, you will have to use the corresponding command to download the toolchain:


// 64-bit systems ->
pacman -S --needed base-devel mingw-w64-ucrt-x86_64-toolchain

// 32-bit systems ->
pacman -S --needed base-devel mingw-w64-ucrt-x86_64-toolchain

Press Enter to accept the default number of packages for the toolchain and then enter Y when you are asked to Proceed with instalation [Y/n]. The toolchain should begin to install and this whole process should take around take around 5 minutes.

Perfect! Now that we have the toolchain installed, we have to tell VS Code how to locate it on our computer. To do this we use environment variables. Go to your settings and search for Edit Environment Variables. We then go to the User Variables section, click edit and add a new variable named C:\msys64\ucrt64\bin.

Now all we have left to do is check if the compiler is installed correctly. Open you command propmpt / powershell and enter the commands gcc --version and g++ --version. When you enter it, if it tells you the version of gcc/g++ built with MSYS2 id installed (mine is 14.2.0) then you have set it up correctly and are ready to go!!

Running Our First Lines of Code

Now that we finally have our VS Code environment set up, we can start coding. The first thing you should always do before starting any code is to make a simple program that prints "Hello, World!" to the console. In this case we will use C. When you are ready, make a new main.c file and copy and paste the following code into it.


#include <stdio.h>

int main() {
	printf("Hello, World!\n");
	return 0;
}

Once you save the file, find the play button in the top right corner of the screen, click the dropdown arrow and click on the option Run C/C++ File. When prompted, you will select C/C++ gcc.exe build and debug active file from the search bar dropdown at the top of the window and your code will begin to compile.

After you select that, your terminal will open and start running some compile scripts. You dont need to worry about any of them until it tells you if your build compiled succesfully. If it did, return to the TERMINAL tab and there you should see it printed the text "Hello, World!"

Congratulations, you have just run your first program with VS Code! The future possibilities are endless!
